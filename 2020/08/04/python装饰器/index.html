装饰器机制：

1.被装饰函数函数名作为参数传给语法糖@修饰的类。

2.然后，被装饰函数函数名指向装饰器这个类的实例对象。

3.为了防止原本被装饰函数的功能的消失，装饰器类中会返回原本的函数，具体机制以下面代码为例，foo（）被装饰后foo指向counter的实例对象，这时候调用foo（1，2，3）就是调用实例对象的__call__（self,*a,**k）函数,__call__（self,*a,**k）函数执行完附加的功能后会返回原本的foo()函数的功能，也就是被self.func接收的功能。

注意：（self,*a,**k）既能接收参数，也能传递参数
